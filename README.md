###01. Requirement
```
sign-up on expo.io
install expo application from google store
```

###02.Install
```
git clone https://github.com/thorpe/expo-react-app-demo.git
cd expo-react-app-demo
npm install
```

###03.Run
```
npm run start
```

###04.Build And Deploy
```
expo build:ios
expo build:android
```
